1.1.7 Video Explore Device Configuration Using the CLI (console)

        configuring devices from the CLI using Cisco's Internetwork Operating System

     router and switches use CLI often -- 
     device opens in 'User Exec' mode 
       'enable'              = enter privilage mode   
       '?'                   = get list of commands available in current mode
       'configure terminal'  = enter configuration mode to make configuration changes to the device
       'end'                 = leave configuration mode
           after making any configuration changes you MUST save the changes for the next startup
       'copy running-config startup-config'
