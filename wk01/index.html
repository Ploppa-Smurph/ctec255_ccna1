1.2.1 Host Roles

    email server 
    web server
    file server

    clients request information from Hosts/servers

1.2.2 Peer-to-Peer

    small networking setup == not as scalable, not as secure

    easier to setup, but not as centralized

1.2.3 End Devices

    client machines, smart tv
    smartphones, tablets
    laptops and pcs

1.2.4 Intermediary Devices

    devices you connect through -- connects end devices
    wireless router
    router
    LAN switch - layer 2 switch
    Multilayer switch - layer 3 switch
    Firewall
    

    management of data as it flows through a network is the role of the intermediary device
     - Regenerate and retransmit data signals
     - Maintain information about what pathways exist in the network
     - Notify other devices of errors and communication failures

1.2.5 Network Media

       copper cable       -- electrical impulse  (cat6; cat7; coaxial cable)     (mid security)
       glass/fiber-optic  -- pulses of light                                     (MOST secure)
       wireless           -- modulation of frequencies of electromagnetic waves  (Least secure)
