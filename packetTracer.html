1.1.3 GUI and CLI Configuration in Cisco Packet Tracer

      Physical tab: physical interface with device

      Config tab: many devices can be configure GUI or through CLI using the IOS (Internetwork Operating System)

      CLI tab: access to Command Line for Cisco Internetwork Operating System 

      Desktop tab: access to IP configuration, Wireless config, command prompt, web browser, etc 

      Services tab: allows configuration of HTTP, DHCP, DNS, etc server processes

2.0 Packet Tracer File Types


2.0.1 Cisco Packet Tracer File Types

      .pka  - packet tracer activity -- has instruction window and is scored
      .pkt  - simulated network is built and saved, not scored 
      .pksz - PTTA (tutored activity file) -- has media, contextualized hints, and a script 
      .pkz  - available, but deprecated -- now use .pkt or .pka file


2.0.2 Cisco Packet Tracer Assessments

      PTMO -- investigate a network and answer questions
      PTSA -- standalone skills assessment during mid or end of course
