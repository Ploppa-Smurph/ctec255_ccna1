Review Questions

01) A teacher is having difficulties connecting his PC to the classroom network. He needs to verify that a default gateway is configured correctly. What is the best command to use to accomplish the task?
    ipconfig

02) Which attack involves a compromise of data that occurs between two end points?
    man-in-the-middle attack
   
03) Match the type of information security threat to the scenario. (Not all options are used.)
    information theft  - obtain trade secrets 
       identity theft     - pretend to be someone else by using stolen personal information
       data loss          - install virus code to destroy recording 
       disruption service - prevent users from accessing a website by sending large number of links   
    
04) What feature of SSH makes it more secure than Telnet for a device management connection?
    login information and data encryption
    
05) Match the description to the type of firewall filtering. (Not all options are used.)
    Stateful Packet Inspection - based on whether traffic is in response to request from internal host
       URL filtering              - based on web address or keyword
       application filtering      - based on port numbers
       packet filtering           - based on mac / ip address    
    
06) What is the purpose of the network security authentication function?
    require users to prove who they are
    
07) Match each weakness with an example. (Not all options are used.)
    technological weakness    - examine os of network device for vulnerabilities 
       configuration weakness    - network engineer did not filter a type of malicious traffic.
       security policy weakeness - did not consider implications or unauthorized users
    
08) What is the role of an IPS?
    detecting and blocking of attacks in realtime

09) What is the advantage of using SSH over Telnet?    
    ssh provides secure communication

10) What is the difference between a virus and a worm?
    worms self-replicate but viruses do not
    
11) Which firewall feature is used to ensure that packets coming into a network are legitimate responses to requests initiated from internal hosts?    
    stateful packet inspection
    
12) A user is redesigning a network for a small company and wants to ensure security at a reasonable price. The user deploys a new application-aware 
       firewall with intrusion detection capabilities on the ISP connection. The user installs a second firewall to separate the company network from 
       the public network. Additionally, the user installs an IPS on the internal network of the company. What approach is the user implementing?
    layered
    
13) An administrator decides to use "WhatAreyouwaiting4" as the password on a newly installed router. Which statement applies to the password choice?
    it is strong because it uses a passphrase
    
14) Which statement describes the ping and tracert commands?
    tracert uses each hop, while ping shows a destination reply only
    
15) A technician is to document the current configurations of all network devices in a college, including those in off-site buildings. Which protocol 
       would be best to use to securely access the network devices?
    ssh

16) What is considered the most effective way to mitigate a worm attack?
    Download security updates from the operating system vendor and patch all vulnerable systems.    
    
17) On which two interfaces or ports can security be improved by configuring executive timeouts? (Choose two.)

18) PT activity - Which command has to be configured on the router to complete the SSH configuration?

       ssh line vty commands. What three additional configuration actions have to be performed to complete the SSH configuration? (Choose three.)
    Generate the asymmetric RSA keys, Configure the correct IP domain name, Create a valid local username and password database.
